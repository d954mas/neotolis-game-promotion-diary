## Sampling Rate

- **After every task commit:** Run `pnpm test:unit`
- **After every plan wave:** Run `pnpm lint && pnpm typecheck && pnpm test:unit && pnpm test:integration && pnpm test:browser`
- **Before `/gsd:verify-work`:** Full suite + `bash tests/smoke/self-host.sh` must be green
- **Max feedback latency:** ~2s (unit), ~3 min (full)
