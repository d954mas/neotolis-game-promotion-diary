## Test Infrastructure

| Property | Value |
|----------|-------|

| **Config file** | `vitest.config.ts` (test.projects: `unit` / `integration` / `browser`) |
| **Quick run command** | `pnpm test:unit` |
| **Full suite command** | `pnpm test` (paraglide compile + svelte-kit sync + every project) **plus** `pnpm test:browser` (needs vite preview on :5173) |
| **Estimated runtime** | ~50s unit + integration; ~30s browser (Phase 2.1 adds ~5s for new feed/sources/dismissal tests) |

---

## Sampling Rate

- **After every task commit:** Run `pnpm test:unit` (~5s)
- **After every plan wave:** Run `pnpm test:integration` (~50s; needs Postgres)
- **Before `/gsd:verify-work`:** `pnpm test` + `pnpm test:browser` + smoke job all green
- **Max feedback latency:** ~5s (unit) / ~50s (integration)
