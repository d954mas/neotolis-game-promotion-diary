## Accomplishments

- **/events/[id]/edit standalone toggle.** Form gains a "Not game-related" checkbox bound to metadata.triage.standalone. Submit handler fires PATCH /api/events/:id (main fields, existing) followed by PATCH /api/events/:id/mark-standalone OR /unmark-standalone — but only when the toggle's value differs from the loaded value. Two PATCHes preserves the dedicated audit verb forensics (event.marked_standalone / event.unmarked_standalone) and lets the conflict-guard 422 fire correctly.
- **/events/[id]/edit UI conflict guard.** $derived `standaloneConflict = editStandalone === true && hasAttachedGame` disables Save AND surfaces an inline error when the user tries to mark a game-attached event as standalone. Plan 02.1-28's service-layer 422 standalone_conflicts_with_game is defense-in-depth.
- **/events/[id]/edit Delete button at footer.** Form footer carries a destructive-color Delete button. ConfirmDialog flow + soft-delete + restore-within-60-days semantics. Replaces the /events/[id] read-only Delete.
- **/events/[id] read-only detail page reshaped.** Edit pencil moves to a 44x44 top-right corner (position: absolute, var(--space-md) inset). Delete button REMOVED entirely (now only at /events/[id]/edit footer). Restore button stays for the ?deleted=1 view. ConfirmDialog import + delete handlers all stripped.
- **AttachToGamePicker compact mode.** New compact?: boolean prop. When compact, the trigger swaps to m.feed_card_attach_compact_label() ("Attach") and applies class:compact for shrunken visual (smaller font, transparent background, tighter padding, muted color). Expanded dropdown unchanged.
- **FeedCard picker visibility gate.** Picker-line wrapper now gated on isInboxRow (existing Plan 02.1-24 derivation). Cards with gameIds.length > 0 OR metadata.triage.standalone === true hide the picker entirely. Inbox cards pass compact={true} to AttachToGamePicker.
- **6 new Paraglide keys** (event_edit_aria_label, events_edit_delete_button, events_edit_standalone_label, events_edit_standalone_help, events_edit_standalone_conflict, feed_card_attach_compact_label). EXPECTED_KEYS in tests/unit/paraglide.test.ts extended in lock-step. messages/en.json values landed via Plan 02.1-31's parallel executor (commit 37d36d7).
- **3 new integration tests** (Plan 02.1-32 describe block in tests/integration/events.test.ts) — round-trip PATCH + mark-standalone reaches metadata.triage.standalone=true with audit chain, conflict 422 defense-in-depth, DELETE then 404.
- **7 new SSR-render-time regression guards** (Plan 02.1-32 describe block in tests/integration/audit-render.test.ts) — picker visibility gate (3 cases), compact label swap (2 cases), /events/[id] edit-pencil presence + Delete absence, /events/[id]/edit standalone toggle + Delete-at-footer wiring.
